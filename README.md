# Bouncing Balls Example
The quickest way to view this project is to visit https://bouncingball.ridhwan.io.

# Project Set Up
After cloning/downloading this project, follow these steps to get started.

Pre-requisites
- Node.js

Run these commands to set up and build the project
```
npm install
npm run build
```

This should install the required packages to build this project.
Run the project by opening `/dist/index.html`

Libraries used in this project:
- `css-loader` used in conjuction with style-loader to load css file into index.js
- `style-loader` used in conjuction with cc-loader to load css file into index.js
- `html-webpack-plugin` used by webpack to generate index.html file
- `webpack` used to compress and bundle source files.
- `webpack-cli` used by build script to run webpack command
